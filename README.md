# Biblioteca Online

Um site simples e elegante de biblioteca virtual desenvolvido com ASP.NET Core MVC.  
O projeto permite visualizar livros, realizar buscas e ver detalhes completos de cada obra.

## Sobre o Projeto

A Biblioteca Online foi criada para oferecer uma interface clean e funcional para explorar livros.  
O design utiliza tons de roxo e possui navegação intuitiva.

### Funcionalidades

- Lista de livros com capas
- Busca em tempo real por título ou autor
- Visualização de detalhes ao clicar em um livro (resumo, ano, gênero e informações)
- Clique no nome do autor para ver mais informações sobre ele
- Interface responsiva

### Livros incluídos

- O Inferno - Dante Alighieri
- Dom Casmurro - Machado de Assis
- As 48 Leis do Poder - Robert Greene
- A Empregada - Freida McFadden

## Tecnologias Utilizadas

- ASP.NET Core MVC
- C#
- HTML5, CSS3 e JavaScript
- Razor Views
